# Virtuals 활용 루트

Chrome 로그인 세션에서 `https://app.virtuals.io/`를 열어 확인할 항목입니다. 2026-06-02 공개 문서와 검색 결과 기준으로는 Virtuals가 AI agent commerce, agent launch, tokenization, ACP offerings, referral/reward, Genesis point 흐름을 제공합니다.

## 확인 순서

1. `app.virtuals.io`에서 Google 로그인 상태 확인
2. 지갑/네트워크 연결 상태 확인
3. ACP, Agents, Offerings, Jobs, Rewards, Referral 같은 메뉴 존재 여부 확인
4. "작업을 수행하는 agent" 등록 또는 offering 생성이 가능한지 확인
5. 결제/정산 토큰이 SOL인지, VIRTUAL/Base 계열인지 분리해서 기록

## 현재 작업대와 연결할 offering

```text
Small scoped AI work agent:
- Android emulator QA report
- Browser game prototype/playtest report
- AI output verification and failure taxonomy
- Evidence-first delivery: screenshots, logs, reproduction steps
- Preferred settlement wallet for SOL jobs: 2ngZYnmBNJNvJsxupQLE1j5GhdKLZfAHse1BkgDxBwWD
```

## 주의

- Virtuals 자체는 Solana 전용 플랫폼이 아니므로, SOL 목표와 바로 일치하지 않을 수 있습니다.
- 투기성 토큰 매수/매도는 5 SOL 수익 작업과 분리합니다.
- 실제 등록, 지갑 연결, 서명, 결제 요청, 약관 동의는 사용자 확인 후에만 진행합니다.
